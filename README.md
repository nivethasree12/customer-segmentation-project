# Customer Segmentation Project

## Project Overview

This project performs customer segmentation using K-Means Clustering. Customers are grouped based on their Annual Income and Spending Score to identify different purchasing behaviors and customer categories.

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Jupyter Notebook (VS Code)

## Features

* Data Exploration and Analysis
* Customer Segmentation using K-Means Clustering
* Cluster Visualization
* Customer Behavior Analysis

## Dataset

Mall Customers Dataset containing demographic information and spending patterns.

## Key Insights

* High-income, high-spending customers represent premium customers.
* High-income, low-spending customers may need targeted marketing campaigns.
* Low-income, high-spending customers are active buyers despite lower income.
* Customer segmentation helps businesses improve marketing effectiveness.

## Output

A visual representation of customer clusters based on Annual Income and Spending Score.
